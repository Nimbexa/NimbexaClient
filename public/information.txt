 HolaClient Theme setup guide.

 For Users:
 Please refer the docs given by the appropriate theme maker!

 For Developers:
 HolaClient structure
 |- bot
 |- database
 |- logs
 |- public [STATIC]
 |- routes
 |- uploads
 |- views [VIEWS]
 |- settings.json
 |- package.json
 |_ index.js

 To serve your theme's file as static please make sure your `views/{themename}`
 matches with `public/{themename}`. Its case sensitive. Everything else is 
 easy to make like 'pages.json' etc. To get more help please contact us on 
 our Discord, https://discord.gg/CvqRH9TrYK
